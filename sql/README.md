# SQL Analysis

## Overview

This folder contains the SQL queries used to analyze the cleaned Texas county expenditure dataset. These queries are designed to explore spending patterns across counties, agencies, and major spending categories.

The dataset used for these queries is the cleaned dataset generated by the Python data cleaning script.

Dataset source:

data/cleaned/texas_county_expenditures_cleaned.csv

The SQL analysis focuses on identifying spending trends and comparing how expenditures are distributed across counties and government agencies.

---

## SQL File

The main SQL script for this project is:

sql/expenditure_analysis_queries.sql

This file contains multiple queries used for analysis and visualization in Power BI.

---

## Queries Included

### 1. Total Spending by County

Calculates the total amount of spending associated with each Texas county.

This query excludes records labeled as `UNKNOWN`, which represent expenditures not tied to a specific county.

Example insight:
Identify which counties receive the highest levels of spending.

---

### 2. Total Spending by Major Spending Category

Aggregates expenditures by major spending category.

This query helps show how government funds are distributed across different types of programs or services.

Example insight:
Determine which spending categories account for the largest share of expenditures.

---

### 3. Total Spending by Agency

Calculates total spending associated with each government agency.

This query helps identify which agencies are responsible for the largest expenditures.

Example insight:
Compare spending levels across agencies.

---

### 4. Spending by County and Major Spending Category

Breaks down spending within each county by major spending category.

This query provides a more detailed view of how expenditures are allocated within each county.

Example insight:
Identify the spending priorities of individual counties.

---

### 5. County Share of Total Spending

Calculates the percentage of total county-level spending represented by each county.

This helps highlight how spending is distributed across Texas counties.

Example insight:
Determine which counties account for the largest share of total spending.

---

## Notes

Some records contain `UNKNOWN` in the county field. These represent expenditures that are not tied to a specific Texas county, such as statewide programs or administrative costs.

For county-level comparisons, these records were excluded from most queries.

---

## Usage

These queries can be run in any SQL environment after importing the cleaned dataset into a table named:

expenditures

The results of these queries are used to support the visualizations built in Power BI.
